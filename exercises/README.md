# Java Practice Exercises

This folder contains standalone Java exercises derived from Warp Drive's search core.

The exercises are intentionally separate from production code under `src/main` and are not part of the Spring API.

## Current exercises

- `01-ngram-tokenizer` — Unicode code points, sliding windows, validation
- `02-inverted-index` — maps, lists, grouping, postings, replacement
- `03-search-engine` — composition, scoring, sorting, and limits

Future Java practice exercises should be added here in their own numbered folder.
